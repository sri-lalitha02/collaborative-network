# Collaborative Network

A full-stack collaborative platform for team management, project tracking, task management, notifications, and real-time communication.

## Repository Structure

```text
collaborative-network/
└── backend/
```

### Backend

The backend API source code is located in the `backend` directory.

Features:

* JWT Authentication & Authorization
* Team Management
* Project Management
* Task Tracking
* Notifications
* Collaboration Requests
* Dashboard Analytics
* Socket.IO Real-Time Communication
* Swagger API Documentation
* Role-Based Access Control (RBAC)
